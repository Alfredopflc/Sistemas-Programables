![Evidencia 2](../img/C5.1_Evidencia7.png)

3. Agregue el programa creado para lograr la condición solicitada.
   
```arduino
int LDR = 34; 
int LDR_val = 0;

void setup() {
  Serial.begin(9600);
}

void loop() {
  LDR_val = analogRead(LDR);
  Serial.println(LDR_val);
   
  delay(1000);
}
```
